# Roadmap

## Near Term

- Add local `.env` loading for easier developer setup.
- Improve submission instructions with a stricter example format.
- Add better error handling for unsupported message types.

## Mid Term

- Store submissions in a persistent database.
- Add moderator commands for reviewing or tagging entries.
- Add logging and basic monitoring for deployments.

## Long Term

- Support multiple events from one bot.
- Add a lightweight admin panel for organizers.
- Export submissions for judging workflows.
